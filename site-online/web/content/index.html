Web3 JS Builder

For the busy solidity developer

Go from an ABI file like this:

[ { "inputs": [], "name": "numberOfMints", "outputs": [ { "internalType": "uint256", "name": "", "type": "uint256" } ], "stateMutability": "view", "type": "function" }]

To JS functions like this:

chain.read.MyContract.numberOfMints() .then(function(mint_count){ alert(mint_count); }) .catch(function(err){ console.error(err); });

Complete with multi-wallet support, WalletConnect integration and web3.js objects.

Conversion takes place client-side in JavaScript. Your ABI is bundled into the JS to avoid multiple network calls.
